# PoorMansThrottle — DIY Firmware Repository

This repository contains **prebuilt ESP32-WROOM firmware binaries** and step-by-step DIY instructions for the **PoorMansThrottle** product.

⚠️ This repository does NOT contain source code.

## Repository Structure

- `/docs` → Instructions and guides
- `/docs/assets` → Images and screenshots
- `/firmware` → Compiled ESP32 firmware binaries

## How to Use

1. Go to the `/docs` folder for flashing instructions.
2. Download the latest firmware from `/firmware`.
3. Follow the flashing guide carefully.

## Disclaimer

Flashing firmware incorrectly may permanently disable your ESP32 device. Proceed at your own risk.
