Autonomous Scheduling

Now you can schedule autonomous runs.

The scheduling screen lets you automate when your locomotive starts, when it stops, and which commands are sent at the beginning and end of the run. It is a simple way to create repeatable operation without needing manual input every time.

Set the days, times, and commands that fit your display

You can now schedule an autonomous run of your locomotive directly from the configuration screen. Choose the days the schedule should be active, set the start time and stop time, and define the command that begins the run as well as the command that ends it. This gives you much more control over how your locomotive starts and how it comes to a stop.

Instead of relying on the same fixed behavior every time, scheduling allows you to tailor operation to the environment. You can decide exactly when the locomotive should begin moving, how long it should run, and what stop behavior is most appropriate for the display or venue.

Choose active days Enable the schedule only on the days you want, whether that is every day, weekends, or a custom weekday pattern.
Define start and stop times Set the exact time the scheduled run should begin and the exact time it should end.
Customize the ON command Select the command that starts the scheduled run so the locomotive begins in the way that best suits your setup.
Customize the OFF command Choose the command that ends the schedule so the locomotive stops the way you want it to stop.
This makes the locomotive more adaptable for displays that need consistent, hands-off operation throughout the day.

It is especially useful for holiday train gardens, seasonal displays, bars, restaurants, retail spaces, or any business that features a train display and wants reliable scheduled movement without constant supervision. With scheduling in place, the locomotive can follow a predictable routine that matches your hours, your event, or your audience.

The result is a cleaner experience for operators and a more polished presentation for viewers. Once configured, the locomotive can start and stop on schedule with behavior tailored to your needs, helping your display feel intentional, smooth, and professionally managed.
