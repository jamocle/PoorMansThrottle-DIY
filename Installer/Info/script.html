Scripting

PMT now supports autonomous scripting.

Scripting makes it possible to run the train autonomously through scripted commands. The scripting language is continuing to expand, allowing for more and more logic over time.

How to access the script page

From the throttle terminal, press and hold the Send button. A normal tap sends the current command, but a long-press opens the dedicated script page where larger scripted sequences can be entered and run.

The script page is intended for more than one-off commands. It provides a place to enter a sequence of instructions that can direct train behavior automatically, helping reduce repetitive manual control when a repeatable flow is needed.

Step 1 Open the throttle terminal and locate the Send button.
Step 2 Long-press Send to open the Run Script page.
Step 3 Enter the script in the editor area.
Step 4 Run the script to let the train follow the scripted command sequence.
Scripting is designed for autonomous control through commands, and the language is growing over time to support increasingly capable logic and more advanced scripted behavior.

As the language expands, scripting becomes more useful for creating repeatable train actions, more structured command sequences, and richer control flows without requiring every step to be handled manually in real time.
