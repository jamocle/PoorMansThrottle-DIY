Throttle UX Update

Faster navigation. Faster stopping.

The throttle screen now supports horizontal swiping between connected throttles and introduces a long-press emergency stop action that can immediately alert all connected throttles.

Swipe left or right to move between connected throttles

The throttle screen now allows horizontal swiping, making it much easier to move between connected throttles directly from the throttle view itself. This removes the old requirement to return to the Scan screen every time you wanted to switch to another connected throttle.

Two chevrons are now shown in the top bar to make that navigation more obvious and more convenient. These chevrons give the user a clear visual cue that other connected throttles are available and can be reached quickly without breaking workflow or backing out of the active screen.

Horizontal swipe navigation Users can swipe across the throttle screen to move between connected throttles in a more natural mobile gesture.
Chevron shortcuts The left and right chevrons provide an easy tap target for moving to the previous or next connected throttle.
Less interruption There is no longer a need to return to the Scan screen just to access another throttle that is already connected.
Better flow This keeps the operator focused on control and status instead of repeatedly stepping out to navigate.
This update is designed to reduce friction. Navigation between connected throttles is now immediate, discoverable, and much more efficient during active use.

Long-press any stop button to send an Emergency Stop

A long-press on any stop button now triggers an E-STOP notification to all connected throttles. This allows the system to rapidly communicate a stop condition across the active throttles and bring them all to a quick stop.

This behavior is intended to make emergency response more immediate and more consistent. Instead of treating the stop button as only a local action, the long-press now acts as a broader safety control that informs every connected throttle of an emergency stop event.

Long-press action Pressing and holding any stop button activates the emergency stop behavior instead of a normal single-stop action.
Broadcast to connected throttles All connected throttles are notified so the stop condition is shared across the active session.
Quick stop response Connected throttles are brought to a rapid stop to support safer handling during critical moments.
Simple and memorable The gesture is easy to remember because it builds directly on a control users already know.
The emergency stop behavior is specifically tied to a long-press, helping distinguish it from ordinary stop usage while still keeping it quickly accessible when it matters most.

Together, these changes improve both usability and operational confidence. The throttle screen is now easier to navigate during normal use and more capable during urgent situations, all while keeping the interface clean, mobile-friendly, and straightforward.
