Beta for Android

MU Locomotive Consist has arrived in PMT.

Multiple-unit control is the railroad practice of running more than one locomotive from a single control position, with the grouped locomotives operating together as one consist. In North American usage, that grouped set of locomotives is commonly called a consist. PMT now brings that idea into the app, so you can build a consist and run multiple engines with one control — like the big boys.

What MU Locomotive Consist means in the app

In railroad terms, multiple-unit control allows the lead locomotive to control the traction behavior of additional locomotives so they work together as a single operating group. PMT now supports that same operating concept in beta for Android by letting you create a consist inside the app and then control the grouped locomotives together from one place.

That means less jumping between separate throttles and a much more natural way to handle multi-engine operation. Instead of treating each locomotive as a completely isolated control session, you can organize them into one consist and operate them as a coordinated unit.

One control for multiple engines Group multiple locomotives together and run them as one operating set instead of controlling each engine separately.
Built in the app The consist can be created and managed directly inside PMT without needing a separate workflow outside the app.
Direction per locomotive Each locomotive in the consist can be set for forward or backward orientation so the consist matches real-world lash-up needs.
Easy editing Locomotives can be added, reordered, reviewed, and adjusted as part of the consist editing flow.

Create and configure your consist

The supplied screens show the new consist workflow: start from the Scan screen, create a locomotive MU consist, then edit the consist name, add locomotives, arrange their order, and set the direction of each locomotive in the group.

This is a major usability step for Android users. MU operation is all about making several locomotives act together from one control point, and PMT now lets you configure that directly in the app with a clean, guided workflow.
