Configuration Backup and Restore

Save settings. Restore them later. Duplicate them with ease.

Throttle settings can now be copied out of the device, saved as a backup, and pasted back later. The same copied settings can also be applied to another throttle, making it much easier to duplicate configuration across multiple throttles without manually re-entering every value.

How the backup and copy workflow works

The screens below show the full path: copy the CV values from the Configuration page, save that copied content, then paste it into the script screen and send it to restore settings or duplicate them onto another throttle.

From the Configuration screen, the Copy action gathers the throttle's CV settings and places them on the clipboard. That copied output can be saved somewhere safe as a backup, such as in a note, document, or maintenance record. Later, if the throttle needs to be restored, those same values can be pasted back into the app.

This same workflow is also useful when you want multiple throttles to share the same configuration. Instead of manually repeating the same CV setup one value at a time, you can copy the settings from one throttle and paste them onto another throttle to duplicate the configuration much more quickly and consistently.

Simple backup method Settings can be copied off the device so they are not lost if a throttle needs to be reset or rebuilt later.
Faster duplication A known-good configuration can be copied from one throttle and reused on another without manually entering each CV.
Better consistency Reusing the same copied script helps keep multiple throttles aligned with the same settings.
Practical field workflow The clipboard-based process makes it easy to move configurations between screens and devices as needed.
In short, this feature turns throttle configuration into something you can preserve and reuse. You can back up settings for safekeeping, restore them later if needed, or paste them into another throttle to duplicate the same setup across more than one unit.
