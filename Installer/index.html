Firmware Installation

Recommended: Install Latest

Use Chrome or Edge (Safari/Firefox won’t work for flashing). Use a data-capable USB cable.

If it connects but doesn’t start flashing: hold BOOT, click Install, select the port, then release BOOT once “Erasing/Writing” begins.

Install an older firmware (optional)

Useful if you want to roll back after testing a new release.

Troubleshooting

  • Close Arduino Serial Monitor or anything else using the COM port.
  • Windows: CP2102 usually works automatically; if not, install Silicon Labs CP210x driver.
  • If you see endless resets, re-flash using “Latest” again.
