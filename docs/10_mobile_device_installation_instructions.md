## What you are downloading

You are downloading the Android app as an **APK** file.

An APK is the install file Android uses for apps that are downloaded from a website instead of an app store.

## Before you start

Please make sure:

- You are using an **Android phone or tablet**
- Your device has enough battery
- You have an internet connection
- You downloaded the app from the official Poor Man’s Throttle website

## Step 1 - Download the APK

1. On your Android device, open the Poor Man’s Throttle website.
2. Open **App Installation**.
3. Open **Android**.
4. Tap **Download Latest Android Version**  
   or tap the **Download Here** button on this page.
5. Wait for the APK download to finish.

After the file downloads, Android may show a download notification, or you may need to open your browser downloads list.

## Step 2 - Open the downloaded file

After the APK finishes downloading, do one of these:

- Tap the **download notification**
- Open your browser menu and go to **Downloads**
- Open the **Files**, **My Files**, or **Downloads** app and find the APK file

Then tap the APK file.

## Step 3 - If Android blocks the install

The first time you install an app downloaded from the web, Android may block it until you allow installs from that source.

You may see a message like:

- **For your security, your phone is not allowed to install unknown apps from this source**
- **Install unknown apps**
- **Allow from this source**

If you see that kind of message:

1. Tap **Settings**
2. Turn on **Allow from this source**  
   or **Allow app installs**
3. Go back
4. Tap the APK file again if needed

This setting is usually applied to the app you used to open the file, such as:

- Chrome
- Edge
- Samsung Internet
- Files
- My Files

## Step 4 - Install the app

Once Android allows installation:

1. Tap **Install**
2. Wait for the installation to complete
3. Tap **Open** to start the app

You can also tap **Done** and open the app later from your app list.

## Step 5 - First launch

When you open the app for the first time:

1. Read any prompts shown by the app
2. Allow permissions if the app needs them for normal operation
3. Continue with your hardware setup and app usage

## Updating later

When a new Android version is released:

1. Download the newest APK
2. Open the new APK
3. Tap **Update** if Android asks

If Android does not allow the update, it usually means the old app and new app do not match correctly. In that case, remove the old app and then install the new one.

## Troubleshooting

### The file does not download
- Try the button again
- Make sure your internet connection is working
- Try Chrome or Edge on Android
- Check whether your browser blocks downloads

### I downloaded it but cannot find it
- Check your browser **Downloads**
- Check the **Files**, **My Files**, or **Downloads** app
- Look for a file ending in **.apk**

### Android says it cannot install from this source
- Tap **Settings**
- Turn on **Allow from this source**
- Return and open the APK again

### The Install button is disabled or missing
- Close the installer screen
- Open the APK again from **Downloads**
- Make sure the browser or file manager you used has permission to install apps

### Android gives a warning
Android may warn you when installing apps downloaded from a website.

Only continue if you downloaded the APK from the official Poor Man’s Throttle site.

### The app will not update
- Download the latest APK again
- Open it from **Downloads**
- If update still fails, uninstall the old version and install the new one

4. Tap **Install**
5. Open the app
