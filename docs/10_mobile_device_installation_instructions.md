# Android App Installation Guide

Welcome! This guide shows you **exactly** how to install the Poor Man's Throttle Android app on an Android phone or tablet.

![Android](android.png)

## What you are installing

You are installing the Android app from the official Poor Man's Throttle website by downloading an **APK** file.

- **APK** = the installable app file for Android.
- This is **not** installed through the Google Play Store.
- Because of that, Android may ask you to allow installs from your browser or your Downloads / Files app.

---

## Before you start

Please make sure:

- You are using an **Android phone or tablet**
- You have a working internet connection
- You are on the official Poor Man's Throttle website
- You are ready to allow installation from your browser **if Android asks**

> Tip: The easiest way is to do all of this directly on your Android device.

---

## The exact website flow

From the main website:

1. Open **App Installation**
2. Open **Android**
3. Tap **Android Installation Instructions** if you want to come back to this guide later
4. Tap **Back** to return to the main page
5. Tap **Download Latest Android Version**

That button downloads the latest Android app file:

`poor-mans-throttle-latest.apk`

---

## Step 1 - Download the APK

1. On your Android device, open the Poor Man's Throttle website
2. Expand **App Installation**
3. Expand **Android**
4. Tap **Download Latest Android Version**
5. Wait for the file to finish downloading

### What you may see

Depending on your phone and browser, Android may show one of these:

- a download bar at the bottom of the screen
- a download arrow or notification at the top
- a message asking whether you want to download the file
- a warning that the file came from outside Google Play

If Android asks whether you want to keep or download the file, continue only if you expected this download from the official Poor Man's Throttle site.

---

## Step 2 - Open the downloaded file

After the download finishes, open the APK file.

You can usually do that in **any one** of these ways:

### Option A - From the browser download notification
Tap the completed download notification.

### Option B - From the browser downloads list
Open your browser menu and look for **Downloads**.

### Option C - From the Files or Downloads app
Open your phone's **Files**, **My Files**, or **Downloads** app, then open:

`poor-mans-throttle-latest.apk`

---

## Step 3 - If Android blocks the install

On many Android devices, the first install from a website will be blocked until you allow that source.

### What Android usually says

You may see a message similar to:

- **For your security, your phone is not allowed to install unknown apps from this source**
- **Install unknown apps**
- **Allow from this source**
- **This app was downloaded from your browser** and needs permission to install

This is normal for apps installed outside Google Play.

### What to do

1. Tap **Settings** on that message, if Android shows it
2. Turn on **Allow from this source** for the app that opened the APK

This is usually one of these:

- **Chrome**
- **Edge**
- **Samsung Internet**
- **Files**
- **My Files**
- **Downloads**

3. Go back
4. Open the APK again if needed
5. Continue with the installation

### If you do not see a Settings button

Use your phone Settings and search for:

**Install unknown apps**

On many devices, the path is similar to:

**Settings > Apps > Special app access > Install unknown apps**

Then select the browser or file app you used, and turn on:

**Allow from this source**

> Important: You only need to allow the specific app that is installing the APK. On newer Android versions, this is handled **per app**, not as one global switch.

---

## Step 4 - Install the app

Once Android allows that source:

1. Open the APK again
2. Tap **Install**
3. Wait for the installation to finish
4. Tap **Open**

If you prefer, you can tap **Done** and open the app later from your app list.

---

## Step 5 - First launch

The first time you open the app:

1. Launch **Poor Man's Throttle**
2. Read any on-screen prompts
3. If Android asks for permissions the app needs, review them and allow the ones required for your hardware setup

Depending on how the app works, Android may ask for things like:

- **Bluetooth / Nearby devices**
- **Location**
- other device access needed for connection or control

Only approve permissions that make sense for the app's features.

---

## If Google Play Protect shows a warning

Android may scan apps installed from outside Google Play.

You might see:

- a warning
- a scan request
- a Play Protect message asking you to review the app

### What to do

- If the app is from the official Poor Man's Throttle website and you expected it, review the message carefully
- If Android or Play Protect says the app is **harmful**, **stop** and do not install it until you verify the APK you downloaded is the correct official release
- If the APK is a normal official release and Android simply warns that it is from outside Google Play, follow the on-screen prompts you trust

> Never continue if the file name looks wrong, the website looks wrong, or the warning says the app is dangerous or malicious.

---

## Updating the app later

When a new Android version is released:

1. Return to the website
2. Download the new APK again
3. Open the new APK
4. Tap **Update** if Android asks

For updates to install over the old version, the new APK must be signed with the **same release key** as the version already on your device.

If Android refuses the update because the signatures do not match:

1. Uninstall the old app
2. Install the new APK fresh

> Warning: Uninstalling may remove saved app data, depending on how the app stores its data.

## Troubleshooting

## The file downloaded, but it will not open
Try opening it from the **Files** or **Downloads** app instead of the browser.

## I tapped the APK and nothing happens
Check whether **Allow from this source** is turned on for the browser or file app you used.

## I cannot find the file
Open **Files**, **My Files**, or **Downloads** and look for:

`poor-mans-throttle-latest.apk`

## Android says installation is blocked
Search Settings for:

**Install unknown apps**

Then allow the browser or file app you are using.

## The Install button is disabled or grayed out
Try closing the screen and opening the APK again from your Downloads folder.

## The app will not update
The newly downloaded APK must be signed with the same release key as the installed version.

## Android warns the app is from an unknown source
That is expected for apps installed outside Google Play. Continue only if you downloaded it from the official Poor Man's Throttle website and the file name is correct.

## Play Protect says the app may be harmful
Stop and verify that you downloaded the official APK from the correct site before doing anything else.
